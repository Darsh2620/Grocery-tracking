# Grocery Tracking System (C++)

## Project Overview
This project is a Grocery Tracking System developed in C++. The purpose of this program is to help users manage grocery items by tracking product names, quantities, and frequency of purchases. It solves the problem of manually keeping track of grocery data and provides a simple way to organize and analyze it.

## Features
- track grocery items
- Count frequency of items purchased
- Display all grocery items with quantities
- Search for a specific item
- Simple and user-friendly interface
